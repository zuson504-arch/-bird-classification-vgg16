About AvianAI

AvianAI is a web-based artificial intelligence application that classifies bird species using Transfer Learning with the VGG16 architecture. This project was developed as part of a Deep Learning course assignment.

VGG16 Transfer Learning

This application uses the VGG16 Convolutional Neural Network pre-trained on the ImageNet dataset. Transfer Learning enables the model to learn bird image features efficiently without training from scratch.

  • Input Image Size : 224 × 224
  • Architecture : VGG16
  • Framework : TensorFlow & Keras
  • Optimizer : Adam
  • Loss Function : Categorical Crossentropy
  • Output : 20 Bird Species
